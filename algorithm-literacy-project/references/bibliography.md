# Master Bibliography

This is the consolidated reference list for The Algorithm Literacy Project. Individual documents carry their own References sections; this file gathers the core works in one place, grouped by theme.

We have done our best to give accurate identifiers (DOIs, arXiv IDs, venues). Where we were not fully certain of a stable identifier, we cite author, year, title, and venue without inventing a link. Nothing here is fabricated. If you find an error, please open an issue.

A note on access: many papers are available as open preprints on arXiv even when the published version sits behind a paywall. Search the title plus "arXiv" or "PDF".
