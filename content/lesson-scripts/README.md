# Guided Childhood, the 5 minute lesson explainer scripts

**Written 9 Jul 2026 · 21 lesson scripts, one per module, ages 4 to 18 · Built for the shorts and animation pipeline · No dashes**

These are the voiceover scripts for the 5 minute explainer video that plays before the slides on every lesson. Each is written in the seven beat format from `plans/lesson-explainer-sample.md`: for every beat you get the **Narration** (the words the voice reads), the **On screen** (what the viewer sees), and a paste ready **Illustration prompt** for the image pipeline. Toned for the age it is for, and voiced by the module's host character.

## How to use each script
1. Read the narration as the voiceover, straight into a voice tool or a real read.
2. Generate each beat's picture from its illustration prompt, in the house style, at 16 by 9.
3. Assemble one picture per beat with a gentle move on each still, the character popping in only where the script marks it, cuts on the beat changes.
