How to view as slides
1
Click Open as slides above — the pitch deck opens in your browser as a full interactive presentation.
2
Press F11 (or Cmd+Shift+F on Mac) to go fullscreen for a proper slide experience.
3
Navigate with ← → arrow keys or swipe left/right on a touchscreen.
How to save as PDF
1
Click Print version (pitch deck) or View document — opens the print-ready version.
2
Press Ctrl+P (or Cmd+P). Set destination to Save as PDF.
3
Set margins to None and tick Background graphics. Click Save.
Tip: Always print from the live URL (not a downloaded file) so fonts load correctly and the PDF looks identical to the screen.
